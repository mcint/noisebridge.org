# noisebridge.org

The public website for **Noisebridge** — a volunteer-run hackerspace and 501(c)(3)
nonprofit in San Francisco. This site is the **outreach / donor-relations / "what we
are" / history** face of Noisebridge, distinct from the community wiki (`.net`) and
the tech/experimentation side (`.io`).

## Status: v0.0

A deliberately **barebones but competent** first cut. It has **holes that invite
participation** (the dashed, highlighted blocks) — thin sections are openings, not
oversights. This site belongs to Noisebridge and the people who show up, **not to any
one person**.

Roadmap:
- **v0.0** — this: a bare scaffold, shippable, holes marked.
- **v0.1** — a fuller *scaffold of intent*: a clear public statement of what
  `noisebridge.org` is for, ready to be socialized in Noisebridge and declared openly.

## Run it

Pure static — no build step.

```sh
# just open it
open index.html
# or serve locally
python3 -m http.server   # → http://localhost:8000
```

Files: `index.html` (one page), `style.css`. GitHub-Pages-ready as-is.

## Contributing

Pick a hole and fill it. Bring proposals and disagreements to the Noisebridge
[wiki](https://www.noisebridge.net/) and the open proposal process — the site is meant
to be added to in the open. A concern not adopted for `noisebridge.org` may fit the
community (`.net`) or tech (`.io`) side better.

## Destination

This repo is intended for the Noisebridge GitHub org. **No remote is configured yet —
don't push until v0.1 is ready to be declared publicly.**
